# Responsive Web Design Project

## Overview
This project is a responsive single-page website created for a class assignment.  
It demonstrates basic HTML and CSS techniques to build a layout that adapts to desktop, tablet, and mobile screen sizes.


## Features
1. Responsive Design  
The layout adjusts to all screen sizes by using CSS media queries.

2. Navigation Bar  
A navigation bar with links that scroll to sections of the page.

3. Hero Section  
A hero section, has background image and headline text.

4. Content Section  
A main section showcasing project and dev experience.

5. Footer  
Display copyright and contact information.


## Technologies Used
- HTML5  
- CSS3  
- Media queries for responsive design  

## How to Use
Open or isit the deployed Netlify link: `https://capable-babka-60d8c7.netlify.app/`. Or test locally: open `index.html` file in a web browser.
Resize the browser window to see the responsive behavior.
